## 실출력 검증 상세 체크리스트 (2026-04-29)

### A. 단건(1건) 기준 검증 — 화면 vs PDF vs 이미지
| 항목 | 확인 기준 | 상태 |
|------|-----------|------|
| 페이지 크기 | 화면/출력 모두 A4 비율(794x1123 기준)로 체감 왜곡 없음 | ⬜ |
| 외곽선/경계선 | 외곽선 2px, 행/열 경계선 끊김·겹침 없음 | ⬜ |
| 헤더/본문 비율 | `h-header/h-info/h-body/h-footer` 비율 붕괴 없음 | ⬜ |
| 아이콘/텍스트 정렬 | 아이콘 치우침, 텍스트 수직정렬 틀어짐 없음 | ⬜ |
| 줄바꿈 일관성 | 핵심 문단 줄바꿈 위치가 화면 대비 과도하게 변형되지 않음 | ⬜ |
| 사진 셀 표시 | 잘림/찌그러짐 없이 `contain` 형태로 표시 | ⬜ |
| 서명 영역 | 서명 셀 비율/정렬 정상, 여백 과다·부족 없음 | ⬜ |
| 파일명 규칙 | 이미지 파일명 `TBM_Report_날짜_팀명.jpg` 규칙 준수 | ⬜ |
