- 선제성: 사고 후 대응이 아닌 조기 경보 중심

## 확장 카테고리 — 스마트TBM지휘 (신규)

### 카테고리 목표
- TBM 기록을 넘어, 당일 작업의 **지휘·지시·이행확인** 루프를 완결
- 팀장/관리감독자/안전담당의 역할별 지시를 분리하고 실행 증빙까지 연결
- SafetyDataLab 분석 결과를 현장 지휘 액션으로 즉시 변환

### 카테고리 구성 (v1)
1. **지휘브리핑**: 당일 핵심위험 3건 + 우선 지시사항 자동 제시
2. **지시발령**: 담당팀/담당자/마감시각/확인방식 포함 지시 생성
3. **이행체크**: 지시별 완료/지연/미착수 상태 및 증빙(사진/코멘트) 추적
4. **지휘리포트**: 일일 지휘결과 요약(지시 수, 완료율, 지연 사유) 자동 정리
